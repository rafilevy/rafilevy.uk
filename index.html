oregex

A regex engine - implemented by compiling strings to a regex type which is then converted to a lazily evaluated NFA. The NFA is then be traversed by the search string to find a match.

rcrawl

A command line binary which recursively searches the file tree for a named file or directory. Similar to the linux find utility.
