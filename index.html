My projects

Here's a collection of some noteable programs I've made. For a full list you can check out my Github profile.

oregex

A regex engine created in OCaml. Implemented by compiling strings to a regex type which is then converted to a lazily evaluated NFA. The NFA is then be traversed by the search string to find a match.

rcrawl

A command line program used to recursively search for files or directories built using Rust. Similar to the linux find utility.

EyeOatie

An Arduino library used to create custom IOT devices on ESP8266 based microcontrollers. Devices can read data from sensors or perform actions all controlable from a server.
